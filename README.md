# Quest for the MacBook: CarterPCs Edition 💻🥔

An interactive arcade game and web landing page built specifically to petition **CarterPCs** (Carter Ryan Smith) for a MacBook upgrade. Because asking in TikTok comments gets swallowed by the algorithm.

Live Demo: [https://carterpcs-macbook-game.vercel.app](https://carterpcs-macbook-game.vercel.app)

## 🎮 Game Story & Concept

The developer is currently coding on a literal **12-year-old potato laptop** (Intel Pentium, 4GB RAM, dying battery). The quest is simple: catch falling high-end tech gear on a custom-built mechanical keyboard catcher, dodge the internet trolls, water drops, and fake imposter accounts, and score **1000+ points** to boot up the official **MacBook Petition Decree**.

### Tech Catalog Rules:
*   💻 **MacBook Pro** (`+100 pts`): The ultimate prize. Shiny gold aura. Don't let it touch the ground!
*   ⌨️ **Custom Keyboard** (`+50 pts`): Speed boost and clears liquid damage slow effects.
*   🔌 **Coiled Cable** (`+30 pts`): Sweet aesthetics and quick points.
*   💧 **Water Drop**: Inflicts liquid damage, resetting your combo and slowing your movements down.
*   🤡 **Imposter Carter** (`-100 pts`): Avoid the fake red Carter accounts trying to steal your clout.
*   🔄 **Windows Update**: Freezes your system completely for 1.5 seconds!

## ✨ Features

*   **Sleek Cyberpunk Tech Theme**: Customized HSL glowing colors, custom mechanical keyboard aesthetic catcher, moving radial gradient background, scanlines, and CRT screen effects.
*   **Web Audio API Synth**: Custom retro 8-bit sound effects (coin pick, sweep powerups, freeze signals, victory jingle) generated completely programmatically in the browser. Zero network loading overhead.
*   **Live TikTok Chat Simulator**: Displays funny user comments in real-time based on your playing performance, capturing the creator's TikTok vibe.
*   **Certificate & Social Copier**: When you win, it reveals a verified MacBook Certificate and copies a highly optimized petition text targeting `@carterpcs` to tag him.

## 🛠️ Tech Stack

*   **Core**: HTML5, Vanilla JavaScript, HTML5 Canvas API (60 FPS rendering)
*   **Styling**: Vanilla CSS3 Custom Properties, Nesting, Flexbox/Grid, and modern `@starting-style` transitions
*   **Hosting**: Vercel
